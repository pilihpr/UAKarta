# UAKarta
Interaktiv karta för att enkelt kunna visualisera Regioner och lokalavdelningar inom Unga Aktiesparare.


# Mappen "src"
Filer och nuvarande status: 

**Region Nord**: Kommuner inom regionen, denna fil används inte, men finns kvar ifall det skulle behövas.

**Region Mitt**: Kommuner inom regionen, denna fil används inte, men finns kvar ifall det skulle behövas.

**Region Öst**: Kommuner inom regionen, hämtas i "index.html".

**Region Väst**: Kommuner inom regionen, hämtas i "index.html".

**Region Syd**: Kommuner inom regionen, hämtas i "index.html".

**RegionNord**: Lokalavdelningar inom regionen, kopierad in direkt till "index.html" så denna fil används inte just nu, för testning.

**RegionMitt**: Lokalavdelningar inom regionen, kopierad in direkt till "index.html" så denna fil används inte just nu, för testning.

**swedish_municipalities**: .geojson fil med alla komunner i Sverige. Denna kopieras föra att kombinera polygoner till lokalavdelningar.
